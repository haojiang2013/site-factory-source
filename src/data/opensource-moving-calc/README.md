# Moving Cost Calculator

Free, accurate moving cost estimator JavaScript module. No dependencies.

## Usage



## Features
- Local / Long-distance / International moves
- Packing service cost estimation
- Piano and specialty item surcharges
- Seasonal pricing multipliers
- Storage fee calculation
- Save/load results via localStorage

## Data Source
Cost estimates based on AMSA (American Moving & Storage Association) 2025 data and U-Haul public pricing.
