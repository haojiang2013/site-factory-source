# LootLedger — Free game loot database

## What is LootLedger?

Search real-time game loot data: legendary drop rates (0.5%-10%), loot box prices ($1.99-$9.99), and average monthly spend ($15-$45). Updated regularly.

## Why I Built This

Every game loot database tool I tried had the same problems:

- ❌ "Pages are cluttered with ads"
- ❌ "Hard to compare across games"

So I built LootLedger — a game loot database that actually just works. No email gates, no paywalls, no ads. Just answers.
