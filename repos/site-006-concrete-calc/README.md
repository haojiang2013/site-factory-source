# MixWise — Free concrete calculator

> **Live:** [pourtrue.8zla.com](https://pourtrue.8zla.com/) · **Free** · **No Signup** · **Open Source**

## What is MixWise?

Need a concrete calculator? Estimate cubic yards, bags, and costs for your slab. Includes metric option and waste factor. Based on real data from NRMCA and HomeAdvisor.

## Why I Built This

Every concrete calculator tool I tried had the same problems:

- ❌ "no metric option"
- ❌ "doesnt include delivery"

So I built MixWise — a concrete calculator that actually just works. No email gates, no paywalls, no ads. Just answers.

## Features
