# StatForge — Free game item stats database

> **Live:** [itemarchive.8zla.com](https://itemarchive.8zla.com/) · **Free** · **No Signup** · **Open Source**

## What is StatForge?

Compare weapon stats across games with StatForge's real market data. See pricing for weapon skins, character skins, loot boxes, and in-game currency. Based on Steam Community Market.

## Why I Built This

Every game item stats database tool I tried had the same problems:

- ❌ "No way to see DPS scaling across infusions without opening multiple tabs"
- ❌ "Fextralife ads make scrolling on mobile impossible"

So I built StatForge — a game item stats database that actually just works. No email gates, no paywalls, no ads. Just answers.

## Features

- ✅ Instant results — no waiting
- ✅ No signup — just use it
- ✅ No email required
- ✅ Real data from industry sources
- ✅ Works on mobile
- ✅ Free forever
