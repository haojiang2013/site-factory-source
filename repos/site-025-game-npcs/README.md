# NPCVault — Free game NPC database

## What is NPCVault?

We tested top game NPC databases including NPCVault, D&D Beyond, and World Anvil. Compare starting prices, key features, and typical NPC creation costs based on real data.

## Why I Built This

Every game NPC database tool I tried had the same problems:

- ❌ "Too many ads and popups"
- ❌ "No easy way to get NPC data in JSON"

So I built NPCVault — a game NPC database that actually just works. No email gates, no paywalls, no ads. Just answers.

## Features
