# WeaponWise — Free game weapon comparison

> **Live:** [weaponwise.8zla.com](https://weaponwise.8zla.com/) · **Free** · **No Signup** · **Open Source**

## What is WeaponWise?

Compare three main airsoft rifle types: AEG, spring sniper, and gas blowback. See typical price, FPS, weight, and length specs. Based on real data from major retailers.

## Why I Built This

Every game weapon comparison tool I tried had the same problems:

- ❌ "Perk combinations not supported"
- ❌ "Doesn't factor status effects correctly"

So I built WeaponWise — a game weapon comparison that actually just works. No email gates, no paywalls, no ads. Just answers.
